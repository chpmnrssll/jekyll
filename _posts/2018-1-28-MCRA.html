Overview

Wordpress membership directory plug-in to replace the default user system. Member accounts, profile management, public member directory.

Link
— Julie, Like Minds Design

Requirements

MEMBERS AREA:
  • Each member will have own secure login
  • Admin will be able to change info in any field for any member; members will be able to change some of their own fields
  • Required Fields that Members Can Edit Themselves:
    • First Name
    • Middle Name
    • Last Name
    • Company Name
    • Street Address
    • City
    • State
    • Zip
    • County – selected from list
    • Home Phone
    • Work Phone
    • Cell Phone
    • Fax Number
    • Email
    • Password
    • Company Logo – photo field to upload logo
    • Method of Entry – select from list
    • Software Used
  • Required Fields that Only Admin Will Be Able to Edit:
    • Username
    • Zone – calculated by county selected
    • Type – select from list
    • MCRA Member - drop-down list choose:
      • Participating Member
      • Senior Member
      • Student Member
      • Associate Member
    • Active Member – Yes/ No
    • Waive Fees – check Yes (check to waive all fees, otherwise leave unchecked)
    • Member Fee – calculation based on type of membership
    • Charge Late Fee – check Yes (leave unchecked for no late fees)
    • Issue Decal? - check Yes (check to show a decal was issued)
    • Fees for each level of membership
    • Log payments received
  • Ability to Search Database by:
    • Active Members (unpaid dues)
    • Active Members (paid dues)
    • Inactive Members
    • Nonmembers
  • Report Page – lists all active members with paid dues
  • Admin can export records or reports to Excel
  • Members will have the ability to:
    • Change their own profile data for some fields
    • View their paid dues history
    • View upcoming renewal date
    • View charges due (if any)
  • Website Viewers (including attorneys) will have the ability to:
    • Search directory for a reporter by Name, Company, County, Zone
    • View search results
    • Print search results
